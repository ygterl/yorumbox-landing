Trendyol Entegrasyonu Nasıl Yapılır?

Yorumbox'u kullanmaya başlamak için Trendyol satıcı panelinizden üç bilgiyi almanız yeterli. Adımları takip ederek 5 dakikada kurulumu tamamlayabilirsiniz.

Hemen Başla →
1

ADIM 1 Trendyol satıcı panelinize giriş yapın

partner.trendyol.com adresine giriş yapın.

2

ADIM 2 Entegrasyon bilgileri sayfasına gidin

Hesap Bilgileri → Entegrasyon Bilgileri adımlarını takip edin.

⚠️ Bu ekranda bilgiler boş geliyorsa "Pazaryeri entegratörü ile çalışıyor musunuz?" seçeneğini aktif hale getirin. Çalıştığınız entegratör adına "ikas" yazıp kaydedin.
3

ADIM 3 Üç bilgiyi kopyalayın

Aşağıdaki üç bilgiyi bu ekrandan kopyalayın:

Satıcı ID → Trendyol Satıcı Numarası
API Key → Trendyol API Key
API Secret → Trendyol API Secret
4

ADIM 4 Yorumbox'a yapıştırın ve kaydedin

Admin panelinizde bu bilgileri girin ve kaydedin. Entegrasyon birkaç dakika içinde tamamlanır.

⚠️

Entegrasyonu 4 gün içinde tamamlamazsanız hesabınız otomatik silinir. Uygulamayı yeniden kurarak başlayabilirsiniz.
