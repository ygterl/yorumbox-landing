Genel Sorular

Yorumbox nedir?

Yorumbox, ikas mağaza sahiplerinin Trendyol'daki ürün yorumlarını ikas ürün sayfalarında slider widget olarak göstermesini sağlayan bir SaaS uygulamasıdır. ikas App Store'dan kurulabilir.

ikas Trendyol entegrasyonu nasıl çalışır?

Yorumbox, Trendyol API'si aracılığıyla mağazanızdaki ürünlerin yorumlarını çeker. Siz hangi yorumların gösterileceğini seçersiniz. Seçilen yorumlar ikas ürün sayfalarınızda slider widget olarak görünür.

Hem ikas hem Trendyol hesabım olması şart mı?

Evet. Yorumbox'ın çalışması için aktif bir ikas mağazası ve Trendyol satıcı hesabı gereklidir.

Kurulum ve Teknik Sorular

Kurulum için teknik bilgi gerekiyor mu?

Hayır. Yorumbox ikas mağazanıza otomatik olarak kurulur. Widget scripti mağazanıza otomatik eklenir, herhangi bir kod yazmanız gerekmez.

Kurulum ne kadar sürer?

Yaklaşık 5 dakika. Uygulamayı kurun, Trendyol API bilgilerinizi girin, ürünlerinizi eşleştirin — hepsi bu.

Trendyol API bilgilerini nereden alırım?

Trendyol satıcı panelinizde Hesabım → API Entegrasyonu bölümünden Satıcı No, API Key ve API Secret bilgilerinizi alabilirsiniz. Detaylı rehber için entegrasyon sayfamızı ziyaret edin.

Trendyol API kullanımı ücretli mi?

Hayır. Trendyol API'si tüm satıcılara ücretsiz sunulmaktadır.

Yorumlar ne sıklıkla güncellenir?

Yorumbox haftalık otomatik güncelleme yapar. İstediğinizde admin panelinden manuel güncelleme de yapabilirsiniz.

Widget görünümünü özelleştirebilir miyim?

Evet. Admin panelindeki Ayarlar sekmesinden widget rengini, kaydırma hızını ve yan yana gösterilen yorum sayısını özelleştirebilirsiniz.

Fiyat ve Plan Soruları

Yorumbox ücretsiz mi?

7 gün ücretsiz deneme mevcuttur. Deneme sonrasında ₺399/ay'dan başlayan planlardan birini seçmeniz gerekir.

Planlar arasındaki fark nedir?

Başlangıç (₺399/ay) 100 yorum, Büyüme (₺599/ay) 300 yorum, Profesyonel (₺799/ay) sınırsız yorum gösterimine izin verir. Tüm planlarda widget, fotoğraflı yorumlar ve otomatik kurulum dahildir.

İstediğim zaman iptal edebilir miyim?

Evet. ikas App Store üzerinden istediğiniz zaman iptal edebilirsiniz. Uzun dönem taahhüt yoktur.

Planımı değiştirebilir miyim?

Evet. ikas App Store'daki Yorumbox sayfasından planınızı istediğiniz zaman yükseltebilir veya düşürebilirsiniz.

Hangi ikas planlarıyla uyumludur?

Yorumbox tüm ikas planlarıyla uyumludur.
