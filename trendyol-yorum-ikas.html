Trendyol Yorumlarını ikas Mağazana Nasıl Eklersiniz? Eksiksiz Rehber

Son güncelleme: Nisan 2025 · 8 dakika okuma

Trendyol'da yüzlerce yorum biriktirdiniz. Bu yorumlar ikas mağazanızda da satış yapmanıza yardımcı olabilir. Araştırmalar tüketicilerin %93'ünün satın almadan önce yorumlardan etkilendiğini gösteriyor. Bu rehberde Trendyol yorumlarınızı ikas ürün sayfalarınıza adım adım nasıl ekleyeceğinizi öğreneceksiniz.

Neden Trendyol Yorumları ikas'ta Gösterilmeli?

Sosyal Kanıt Satışları Artırır

Spiegel Research Center araştırmasına göre tüketicilerin %93'ü satın almadan önce yorumları inceliyor. ikas mağazanızda yorum olmadığında ziyaretçiler güvensizlik hissedip ayrılabilir. Trendyol'da zaten biriktirdiğiniz yorumları ikas'ta göstererek bu güven açığını kapatabilirsiniz.

Mevcut Yorumlarınızı İki Kez Kullanın

Trendyol'da yıllar içinde biriktirdiğiniz yorumlar değerli bir varlık. Bu yorumları ikas mağazanızda da göstererek hem zaman kazanırsınız hem de gerçek müşteri deneyimlerini yeni alıcılarla paylaşmış olursunuz.

ikas Mağazanızda Güven Oluşturun

ikas'ta yeni bir mağaza açtığınızda yorum biriktirmek zaman alır. Trendyol'daki mevcut yorumlarınızı aktararak ikas mağazanızı baştan güvenilir gösterebilirsiniz.

Başlamadan Önce Ne Gerekir?

  • ✓ Aktif bir ikas mağazası
  • ✓ Trendyol satıcı hesabı
  • ✓ Trendyol API bilgileri (Satıcı No, API Key, API Secret)
  • ✓ Yorumbox uygulaması (ikas App Store'dan ücretsiz)

Adım Adım Kurulum Rehberi

1

Yorumbox'ı ikas App Store'dan Kurun

ikas yönetici panelinize giriş yapın. Sol menüden Uygulamalar → Uygulama Mağazası'na gidin. Arama kutusuna "Yorumbox" yazın ve uygulamayı bulun. "Ücretsiz Dene" butonuna tıklayarak 7 günlük ücretsiz denemeyi başlatın. Uygulama ikas mağazanıza otomatik olarak kurulur, herhangi bir kod yazmanız gerekmez.

ikas App Store'da Aç →
2

Trendyol API Bilgilerini Girin

Yorumbox'ı kurduktan sonra Ayarlar sekmesine gidin. Trendyol Bağlantısı bölümünde üç bilgi girmeniz gerekir:

  • • Satıcı No: Trendyol satıcı panelinizden alırsınız
  • • API Key: Trendyol → Hesabım → API Entegrasyonu
  • • API Secret: API Key ile aynı sayfadan alırsınız

API bilgilerini nasıl alacağınızı bilmiyorsanız adım adım entegrasyon rehberimizi inceleyin.

3

Trendyol Ürünlerinizi ikas Ürünleriyle Eşleştirin

Ürünler sekmesine gidin. Sol tarafta Trendyol ürünleriniz, sağ tarafta ikas ürünleriniz listelenir. "Otomatik Eşleştir" butonuyla sistem ürün isimlerine göre otomatik eşleştirme yapar. İstediğiniz ürünü manuel olarak da seçebilirsiniz.

4

Göstermek İstediğiniz Yorumları Seçin

Eşleştirdiğiniz ürünün yanındaki "+Yorum Ekle" butonuna tıklayın. Trendyol'dan çekilen tüm yorumlar listelenir. Hangi yorumların gösterileceğini siz seçersiniz. Seçiminizi tamamlayıp "Kaydet" butonuna basın.

5

Yorumlar ikas Mağazanızda Yayında

Yorumlar seçildikten sonra ikas ürün sayfalarınızda otomatik olarak görünmeye başlar. Widget slider formatında "Müşteri Yorumları" başlığıyla ürün sayfanızın alt bölümünde gösterilir. Ayarlar sekmesinden widget rengini ve kaydırma hızını özelleştirebilirsiniz.

Sık Yapılan Hatalar ve Çözümleri

❌ "API bilgileri hatalı" hatası

Trendyol satıcı panelinizde API bilgilerini yeniden oluşturun. Kopyalarken başında veya sonunda boşluk kalmadığından emin olun.

❌ Ürün eşleştirmede yanlış ürün öneriliyor

Otomatik eşleştirme ürün isim benzerliğine göre çalışır. İsimler farklıysa manuel olarak doğru ürünü seçin.

❌ Yorumlar sitede görünmüyor

Yorumları seçip kaydettiğinizden emin olun. ikas ürün sayfasını yenileyin. Hala görünmüyorsa destek@yorumbox.com adresine yazın.

Sık Sorulan Sorular

Trendyol yorumları otomatik güncellenir mi?

Evet. Yorumbox haftalık otomatik güncelleme ile Trendyol'daki yeni yorumlarınızı ikas mağazanıza ekler.

Kaç yorum gösterebilirim?

Seçtiğiniz plana göre değişir. Başlangıç planında 100, Büyüme planında 300, Profesyonel planda sınırsız yorum gösterebilirsiniz.

Trendyol API kullanımı ücretli mi?

Hayır. Trendyol API'si satıcılara ücretsiz olarak sunulmaktadır.

Widget görünümünü özelleştirebilir miyim?

Evet. Ayarlar sekmesinden widget rengini marka renginizle eşleştirebilir, kaydırma hızını ve gösterilen yorum sayısını ayarlayabilirsiniz.

Tüm ikas planlarıyla uyumlu mu?

Evet. Yorumbox tüm ikas planlarıyla uyumludur.
