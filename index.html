Trendyol yorumlarını ikas mağazama nasıl eklerim?

Yorumbox uygulamasını ikas App Store'dan kurun, Trendyol satıcı panelinizden API bilgilerinizi alın ve girin, ardından Trendyol ürünlerinizi ikas ürünleriyle eşleştirin. Göstermek istediğiniz yorumları seçin — kurulum 5 dakika sürer, kod bilgisi gerekmez.

ikas Trendyol entegrasyonu nasıl çalışır?

Yorumbox, Trendyol API'si aracılığıyla mağazanızdaki ürünlerin yorumlarını çeker. Siz hangi yorumların gösterileceğini seçersiniz. Seçtiğiniz yorumlar ikas ürün sayfalarınızda slider widget olarak görünür. Haftalık otomatik güncelleme ile yeni yorumlar otomatik eklenir.

Trendyol API bilgilerimi nereden alırım?

Trendyol satıcı panelinize girin, Hesabım → Mağaza Bilgileri → API Entegrasyonu bölümünden Satıcı No, API Key ve API Secret bilgilerinizi alabilirsiniz. Detaylı rehber için entegrasyon sayfamızı ziyaret edin.

Yorumlar ne sıklıkla güncellenir?

Yorumbox, Trendyol'daki yeni yorumlarınızı haftalık olarak otomatik günceller. İstediğinizde manuel güncelleme de yapabilirsiniz.

Hangi ikas planlarıyla uyumludur?

Yorumbox tüm ikas planlarıyla uyumludur. Yorumbox'ın kendi planları ₺399'dan başlar: Başlangıç (100 yorum), Büyüme (300 yorum), Profesyonel (sınırsız yorum). Tüm planlarda 7 gün ücretsiz deneme mevcuttur.

Kurulum için teknik bilgi gerekiyor mu?

Hayır. Yorumbox ikas mağazanıza otomatik olarak kurulur, herhangi bir kod yazmanız gerekmez. Widget scripti mağazanıza otomatik eklenir.
